# 车辆预定系统

一个基于HTML5的车辆预定Web应用，支持车辆浏览、详情查看和在线预定功能。

## 功能特性

- **车辆列表展示**：卡片式布局展示所有可预定车辆
- **详细信息查看**：查看车辆的完整信息（型号、编码、VIN码、车系等）
- **在线预定**：填写申请原因、选择时间段、输入申请人信息
- **表单验证**：自动验证表单输入和时间范围
- **时间冲突检测**：防止同一车辆在同一时间段被重复预定
- **预定记录管理**：使用LocalStorage保存预定记录
- **响应式设计**：完美适配PC、平板和手机

## 使用说明

### 浏览车辆
1. 打开主页，查看所有可用车辆
2. 每个车辆卡片显示：型号、车系、车辆名称、城市、状态
3. 点击任意卡片或"查看详情"按钮进入详情页

### 预定车辆
1. 在详情页查看车辆完整信息
2. 填写预定表单：
   - **申请原因**：说明预定用途（至少5个字符）
   - **开始时间**：选择预定开始时间（不能早于当前时间）
   - **结束时间**：选择预定结束时间（必须晚于开始时间）
   - **申请人姓名**：输入申请人姓名（至少2个字符）
3. 点击"提交预定"按钮
4. 系统会自动验证：
   - 所有必填项是否填写
   - 时间范围是否合理
   - 是否与其他预定冲突
5. 预定成功后可选择查看预定记录或返回列表

### 查看预定记录
1. 点击顶部导航的"我的预定"链接
2. 查看所有历史预定记录
3. 记录包含：车辆信息、申请原因、时间段、申请人、预定时间

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **存储**：LocalStorage
- **样式**：原生CSS（响应式设计）
- **兼容性**：支持现代浏览器（Chrome, Firefox, Safari, Edge）

- Firefox 88+

- Edge 90+
- 移动端浏览器（iOS Safari, Chrome Mobile）

## 注意事项

1. **数据存储**：所有数据保存在浏览器LocalStorage中，清除浏览器数据会导致预定记录丢失
2. **时间格式**：时间格式为 yyyy/MM/dd HH:mm
3. **预定限制**：单次预定时长不能超过30天
4. **冲突检测**：同一车辆在同一时间段只能被预定一次
